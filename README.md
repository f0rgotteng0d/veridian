# Mission Veridian — Code Execution Guide

## Setup

Ensure Python (≥ 3.8) is installed.

Install required dependencies:

```bash
pip install numpy pandas matplotlib scipy
```

---

## Download the Repository

Clone the repository using:

```bash
git clone https://github.com/f0rgotteng0d/veridian.git
cd veridian
```

Alternatively, download the ZIP and extract it.

---

## Files Required

Make sure the following files are present in the same directory:

* `mission_veridian.py`
* `veridian_ephemeris.csv`

---

## Running the Code

Run the script using:

```bash
python mission_veridian.py
```

---

## Output

The script will:

* Print the full mission analysis in the terminal
* Generate plots and save them in the same directory

---

## Notes

* No manual input is required
* Runtime is approximately **3–4 minutes**
